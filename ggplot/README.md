### <a id="setup">Setup Environment</a>
* Create virtual environment, which will not affect your system-wide installation
  ```
    $ python3 -m venv venv

* Activate virtual environment
  ```
    $ source ./venv/bin/activate

* Install (plotnine)[plotnine reference]
  ```
    $ python3 -m pip install plotnine

* Optional: Install (Jupyter Notebook)[https://jupyter.org/] and activate in venv
  ```
    $ python3 -m pip install jupyter
    $ jupyter-notebook
